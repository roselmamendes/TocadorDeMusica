# TocadorDeMusica
Simple music player.

## Stack

>>AngularJS

>>SoundJS

>>Jasmine

>>Karma

The project has a <b>package.json</b> that is going install the dependencies. For that, use *npm install* (install npm before) in the root of the project.

For run tests, use *npm test*.
